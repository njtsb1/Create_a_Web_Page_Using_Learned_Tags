Module 1 - First Steps with HTML

What is HTML?

HTML has existed since 1991 and is currently in version 5, which is packed with features and functionalities that bring improvements to web development. Its main responsibility is to demarcate the structure of a web page. This HTML structure is made up of a set of elements, that is, hypertexts, which connect to each other forming the page. HTML elements, or also called HTML tags, are used to inform the browser what type of structure is being built, which can be titles, paragraphs, images, links, among others.

What is TAG?

Tags are formed by their own structure, they start with the sign “less than” , then comes the name of that element and finally, the “greater than” sign. They can be arranged in tags that require closing and tags that close alone (self-closing). The closing of a tag will be defined with a slash (/), and in the case of auto tags closure, there is no need for this character to be present.

html

  • Tag used to define an HTML document.

  • This is the root tag of the document structure.

  • With the exception of DOCTYPE, all tags are children or descendants of html tag.
